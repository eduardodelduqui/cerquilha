## Build Setup
If you want to run this application in your localhost:

Clone the repository.

```
git clone https://github.com/eduardodelduqui/cerquilha
```

Install dependecies

```
npm install
```

Run application in your localhost

```
npm run dev
```

Oh, and don't forget [you need Sass installed] in your environment

[you need Sass installed]: https://github.com/sass/sass
